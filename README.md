# MobileApp-Notes
A simple and elegant notes app for Android and iOS, designed to help you write and organize your thoughts anytime, anywhere.

Features
* Basic Notes Management: Create, view, edit, and delete notes effortlessly

* Local Storage: Your data is stored securely on your device
